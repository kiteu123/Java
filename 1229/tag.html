블럭 요소란 ?

가로 너비: 기본적으로 부모 요소의 100% 차지
세로/가로 크기(width, height) 설정 가능
margin, padding 모두 적용 가능
다른 요소를 안에 포함 가능

블럭 요소의 종류 ?

레이아웃 관련 : div, header, nav, main, section, article, aside, footer

텍스트 구조 : h1~h6, p, blockquote, pre

목록 : ul, ol, li, dl, dt, dd

기타 : hr, form, table

인라인 요소란 ?

width / height 적용 안 됨
가로로 나란히 배치됨
margin: 좌우만 적용
padding: 적용되지만 위아래는 시각적 영향 적음
블록 요소를 포함 ❌

인라인 요소의 종류 ?

텍스트 관련 : span, a, strong, em, b, i, mark, small, sub, sup

미디어 : img

기타 : label, input, button
