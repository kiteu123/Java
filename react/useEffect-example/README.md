# useEffect 실전 예제

## 실행 방법

```bash
cd useEffect-example
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 예제 구성

1. **예제1** - 마운트 시 1회 실행 (`[]`)
2. **예제2** - count 변경 시 document.title 변경 (`[count]`)
3. **예제3** - Cleanup (토글 시 언마운트 로그)
4. **예제4** - 통합 (마운트 + 값 변경 + Cleanup)

**콘솔(F12)을 열어서 로그를 확인하세요.**
