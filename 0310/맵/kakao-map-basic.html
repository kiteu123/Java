🗺️ 카카오맵 기본 예제

카카오맵을 웹페이지에 삽입하는 기본 방법입니다.

📍 현재 위치

위도: 37.5665

경도: 126.9780

주소: 서울특별시 중구 세종대로 110 (서울시청)

<!-- 카카오맵 API 스크립트 -->
<script type="text/javascript" src="//dapi.kakao.com/v2/maps/sdk.js?appkey=YOUR_JAVASCRIPT_KEY"></script>

<!-- 지도를 표시할 div -->
<div id="map" style="width:100%;height:500px;"></div>

<script>
    var container = document.getElementById('map');
    var options = {
        center: new kakao.maps.LatLng(37.5665, 126.9780), // 서울시청
        level: 3 // 지도 확대 레벨 (1~14)
    };
    
    var map = new kakao.maps.Map(container, options);
</script>
⚠️ 중요:
1. 카카오 개발자 사이트(https://developers.kakao.com/)에서 JavaScript 키를 발급받아야 합니다.
2. 아래 스크립트의 YOUR_JAVASCRIPT_KEY를 본인의 키로 변경하세요.
3. 플랫폼 설정에서 도메인을 등록해야 합니다.
