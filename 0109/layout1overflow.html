HTML과 CSS 차이

Html은 문서의 뼈대를 만든다. html은 Hyper Text Markup Language 약자이다. 확장자는 html로 지정하면 웹브라우저에서 인식한다.

Css은 글꼴이나, 배경색,너비와 높이, 위치등을 지정한다. css은 cascade style sheet 약자이다. 확장자는 css로 지정하면 웹브라우저에서 인식한다.

JavaScript는 웹 페이지에 동작을 부여한다. 버튼 클릭, 입력 처리, 화면 변경 등을 제어할 수 있다. JavaScript는 웹 브라우저에서 실행되는 프로그래밍 언어이다.

jQuery는 JavaScript를 쉽게 사용하도록 도와주는 라이브러리이다. 복잡한 JavaScript 코드를 간단한 문법으로 작성할 수 있다. DOM 제어와 이벤트 처리를 편리하게 할 수 있다.
